## Overview

---

## Feature(s)

### 1. Opening tickets in new tab on click
- Any ticket opened via a left click will open a new tab instead of opening it on the current page.

---

## Installation

1. Install a userscript manager like [Tampermonkey](https://www.tampermonkey.net/).
2. Create a new userscript and paste the code from `Jira for CSEs.user.js`.  
3. Save and reload Jira pages.
